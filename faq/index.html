Beranda Pedoman MPQ Cek Santri FAQ

Bantuan

Pertanyaan Umum

Beberapa jawaban ringkas terkait data santri, rekom, ADNA, dan layanan digital MPQ Al Falah.

Catatan

Jika data yang tampil belum sesuai, buka detail santri lalu gunakan tombol Laporkan Data Salah.

Apa fungsi Cek Data Santri?
  • Untuk melihat informasi dasar santri MPQ.
  • Menampilkan kode, kelas, ADNA, kamar, muallim, ruang, dan status rekom.
  • Memudahkan pengecekan data tanpa membuka spreadsheet.
Data santri bisa dicari berdasarkan apa?
  • Nama santri.
  • Kelas, contoh: 3 IBT G.
  • ADNA, contoh: A 01.
Kenapa pencarian tidak memakai kode, kamar, atau muallim?
  • Pencarian dibuat sederhana agar hasil lebih tepat.
  • Jika mencari nama muallim, hasil bisa terlalu banyak dan membingungkan.
  • Karena itu pencarian difokuskan pada nama, kelas, dan ADNA.
Apa itu rekom?
  • Rekom adalah tanggungan yang muncul karena santri memiliki catatan alpa sesuai ketentuan MPQ.
  • Rekom 1, Rekom 2, Rekom 3, dan Rekom 4 menunjukkan urutan tanggungan yang perlu diselesaikan.
  • Rincian rekom dapat dilihat pada detail santri jika santri memiliki tanggungan.
Apa arti status Belum dan Selesai?
  • Belum berarti santri masih memiliki tanggungan yang belum selesai.
  • Selesai berarti tanggungan rekom sudah diselesaikan sesuai data yang tercatat.
Apa arti Tidak Rekom?
  • Artinya santri tidak memiliki tanggungan rekom yang perlu diselesaikan.
  • Jika tidak ada tanggungan, bagian rincian rekom biasanya tidak ditampilkan.
Apa itu ADNA?
  • ADNA adalah penanda tingkatan atau kelompok pembelajaran santri di MPQ.
  • Contoh ADNA: A 01, B 02, C 03, atau format lain sesuai data MPQ.
Kenapa nama santri tidak ditemukan?
  • Pastikan penulisan nama tidak terlalu panjang.
  • Coba cari menggunakan kelas atau ADNA.
  • Jika tetap tidak muncul, kemungkinan data belum masuk atau status santri sudah tidak aktif.
Bagaimana jika data santri salah?
  • Buka detail santri.
  • Klik tombol Laporkan Data Salah.
  • WhatsApp admin akan terbuka dengan format laporan otomatis.
  • Tulis bagian data yang salah dan data yang benar.
Apakah data di website ini bersifat final?
  • Data website mengikuti database MPQ yang sedang digunakan.
  • Jika ada perubahan terbaru yang belum tampil, silakan laporkan kepada admin MPQ.
